About Me

Perkenalkan nama saya Ahmad Samudra atau orang biasa panggil saya indra, saya juga memiliki nama panggilan atau nama panggung ASBX ID, Kata ASBX ID berasal dari beberapa singkatan, yaitu ASBX yang singkatan dari (Ahmad Samudra BeatboX), Kenapa beatbox? karena pada jaman SMP saya suka sekali dengan teknik membuat nada instrumen tanpa alat musik hanya besumber dari mulut atau biasa disebut sebagai beatbox, sehingga saya tertarik mempelajarinya hingga mahir, BX selain singkatan dari beatbox saya juga mendapatkan inspirasi nama dari beatboxer terkenal yaitu billy beatbox yang biasa menggunakan nama panggilan BDABX. Selanjutnya singkatan dari ID adalah indonesia.

Saya sejak SMP tertarik dengan beberapa hal yang saya jadikan sebagai hobi saya, seperti bermain gitar dan beatboxing. pada saat masuk ke bangku SMA saya mulai tertarik dengan seni fotografi dan videografi, sehingga pada saat bangku SMA kelas 2 saya memulai project pertama saya untuk sekolah dengan beberapa hasil video cinematic dan sebuah short film, lalu pada saat masuk ke bangku kuliah saya masuk perguruan tinggi dengan jurusan IT, disini saya menjadi sangat tertarik dengan pembuatan sebuah website dan beberapa project di bidang IT seperti robotik, I/O dll, sehingga disini saya sudah menghasilkan beberapa project seperti membuat sistem informasi bookinghotel.com, membuat robot linefollower, membuat alat pendeteksi banjir,dll dan sampai sekarang saya masih mendalami beberapa ilmu yang ada pada jurusan saya
